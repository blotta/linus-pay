# LinusPay

Financial tools

## Tools

### Group Split

- Create groups with one or more users
- Enter and track values each user spent
- Settle remaining amounts


### Price Tracking

- Enter product and service prices you'd like to track over time
- Analyze your own price history
- (Maybe) Enter product link for automatic price tracking on a configurable schedule


### Purchase Calculator

- Compare simulated purchase methods
  - In installments as if amount is invested with a fixed rate
  - One-time payment


### Paper Investments

- Investment simulations
